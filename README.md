# Lotto

## Getting Started

### Run it locally

1. Clone the project

```
git clone git@github.com:RyanLii/Lotto.git
```

3. Install dependencies

```
cd Lotto

# Using npm 
npm install

# Install Cocoapod
sudo gem install cocoapods

cd ios
pod install
```

4. Run the  app
# From Commond line
```
npm start
npm run 
```
# From Xcode
```
open Lotto/ios/RSL.xcworkspace
```
